# 🪙 TOKEN ECONOMICS
### **Extreme Context Optimization for Infinite Sessions**

The **Ironclad Master Framework** enforces **Hardware-Accelerated Token Efficiency**. In an AI-first economy, the context window is your most expensive and fragile resource. We protect it at all costs.

---

## 🎯 Direct Intelligence Targeting
Stop crawling. Start targeting.
- **Protocol**: Never use `ls` or recursive directory searches if the path is known.
- **Rule**: Use `grep_search` with specific `include_pattern` to surgically isolate logic.
- **Outcome**: **100% reduction** in directory listing overhead.

## 🦍 The Caveman Protocol
Brevity is the soul of performance.
- **Protocol**: Activate the `caveman` skill to terminate apologies, preambles, and conversational slop.
- **Rule**: "Implementing X." > "I would be happy to implement X for you today..."
- **Outcome**: **~85% token saving** per conversational turn.

## 📖 The "Pocock" Ubiquitous Language
Semantic compression via `CONTEXT.md`.
- **Protocol**: Define project-specific jargon to represent complex systems with single tokens.
- **Example**: Use "The Aegis" instead of "The decentralized authorization and RBAC verification layer."
- **Outcome**: **Massive context preservation** in long-running architectural shifts.

## ✂️ Surgical Context Loading
- **Micro-Reads**: Use `read_file` with precise `start_line` and `end_line` ranges.
- **Sub-Agent Compression**: Use `@generalist` to digest 2000+ line files and return a **3-sentence semantic summary**.
- **Diff-Only Generation**: Use `replace` for surgical action. Never rewrite what is already perfect.

## 🏁 Goal-Driven Recursion
Transform instructions into autonomous objectives.
- **Format**: `[Action] -> [Verify: Success Condition]`
- **Benefit**: Empowers the agent to loop independently via the **Infinity Harness**, eliminating turn-based latency.
