## 🚀 What is Ironclad?

Ironclad is a production-ready **autonomous business operating system** that combines:

- 🧠 **AI-Powered Quality Assurance** - Self-auditing Truth Score system
- ⚡ **Autonomous Task Execution** - Infinity Harness for goal completion
- 🎯 **Strategic Planning** - SPARC specification generation
- 🔄 **Self-Healing Architecture** - Automatic backup and recovery
- 📊 **Real-Time Monitoring** - Pro dashboard with live metrics
- 🔌 **MCP Integration** - Model Context Protocol for AI agents

## 📦 Installation

```bash
npm install ironclad-master-framework
```

Or use globally:

```bash
npx ironclad --help
```

## ⚡ Quick Start

### 1. Initialize in Your Project

```bash
ironclad init
```

### 2. Run Quality Audit

```bash
ironclad audit
```

Get instant Truth Score analysis across:
- Architecture quality
- Test coverage
- Security vulnerabilities
- Performance bottlenecks
- Accessibility compliance

### 3. Generate Strategic Plans

```bash
ironclad plan "Build AI-powered dashboard"
```

Automatically generates SPARC specifications with context-aware recommendations.

### 4. Autonomous Execution

```bash
ironclad infinity "Achieve 100% test coverage"
```

Launch the Infinity Harness for autonomous goal completion with continuous verification.

## 🔧 Configuration

Create `.ironclad.json` in your project root:

```json
{
  "rules": {
    "designSignature": false
  }
}
```

### Available Rules

- `designSignature` - Require `@ironclad-design-signature` headers in UI files
