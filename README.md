## 🛠️ Quick Start

### 1. Initialize
```bash
git clone https://github.com/scream687/Ironclad-Master-Framework.git
cd Ironclad-Master-Framework
make install
```

### 2. Maintenance
```bash
make upgrade  # Triggers self-evolution
make fetch-skill REPO=user/repo  # Integration remote intelligence
```
