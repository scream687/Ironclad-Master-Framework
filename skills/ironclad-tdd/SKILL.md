# Ironclad TDD Workflow

1. Scaffold failing test suites.
2. Execute the test runner and observe the failure (Red Phase).
3. Implement the minimal logic required to pass (Green Phase).
4. Run the internal Ironclad Truth Audit.
5. Refactor and ensure 0.95+ score.
